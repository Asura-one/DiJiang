# Meta 指南

## 概述

Meta 层项目指南，涵盖项目结构、架构决策、更新日志约定和开发工作流。这些内容约束项目本身如何组织和维护。

## 使用方式

每个指南文件遵循以下格式：

```markdown
# 指南标题

## 目的
此指南存在的原因。

## 规则
可操作规则的列表。

## 示例
优秀模式的实例。

## 反模式
需要避免的常见错误。
```
